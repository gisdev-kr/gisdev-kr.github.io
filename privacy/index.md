이 사이트는 회원가입이나 자체 댓글 데이터베이스를 운영하지 않습니다. 선택적으로 활성화되는 방문 통계와 광고는 각 제공자의 정책에 따라 쿠키 또는 유사 기술을 사용할 수 있습니다.

## 외부 서비스

- GitHub Pages: 정적 사이트 호스팅
- Giscus: GitHub Discussions 기반 댓글(활성화 시)
- Google Analytics / GoatCounter: 방문 통계(설정된 경우)
- Google AdSense: 광고(승인 후 활성화된 경우)

광고 차단 여부와 관계없이 글, 지도, 검색, 다운로드 기능을 제한하지 않습니다. 설정 변경은 이 페이지에 반영합니다.

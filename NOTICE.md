# Design acknowledgements

The site is an original implementation informed by two MIT-licensed Jekyll projects:

- [forestryio/ubuild-jekyll](https://github.com/forestryio/ubuild-jekyll): modular landing-page composition and bold hero treatment.
- [byanko55/jekyll-theme-satellite](https://github.com/byanko55/jekyll-theme-satellite): persistent blog navigation and feature ideas such as theme switching, search, code copy, table of contents, and comments.

No Forestry CMS integration, project branding, demo content, or proprietary service is included.
